## Phase 0: 지금 당장 (오늘)
> 예상 소요: 2~3시간 | 비용: 0원 | 담당: 클로드 + 재웅

### 0-1. FAQ 섹션 추가
- **왜:** 결제 전 궁금한 점을 해소 못 하면 이탈한다
- **내용 (7개):**
  - Q. 어떤 사진을 준비해야 하나요?
  - Q. 옛날 사진을 스마트폰으로 찍어도 되나요?
  - Q. 영상은 어떤 형태로 받나요?
  - Q. 완성까지 얼마나 걸리나요?
  - Q. 환불/취소가 가능한가요?
  - Q. 영상이 마음에 안 들면 어떡하나요?
  - Q. 행사장에서 바로 틀 수 있나요?
- **위치:** 주문 폼 바로 위 (결제 직전 불안 해소)

### 0-2. 카카오 채팅 플로팅 버튼
- **왜:** 실시간 문의 경로가 없으면 불안한 고객이 이탈한다
- **방법:** 카카오 비즈니스 → 채팅 버튼 코드 발급 → index.html에 삽입

### 0-3. 환불/취소 정책 상세화
- **왜:** "영상 불만족 시 재제작 1회"만으로는 부족
- **내용:**
  - 영상 제작 시작 전: 100% 환불
  - 영상 완성 후 불만족: 무료 재제작 1회
  - 재제작 후에도 불만족: 50% 환불
  - 환불 요청: 카카오톡 또는 전화

## Phase 1: 이번 주 (D+1~7)
> 예상 소요: 5~7일 | 비용: ~5만원 | 담당: 재웅

### 1-1. ⭐ 실제 완성 영상 샘플 제작 (최우선)
- **왜:** 이것이 없으면 아무것도 소용없다
- **작업:**
  1. 가족/지인 사진으로 실제 서비스 영상 3~4편 제작
  2. 다양한 케이스: 부부 사진, 가족 단체, 어린시절~현재
  3. MP4 파일로 S3에 업로드 → 자체 video 태그로 재생
  4. 인스타그램 임베드 제거 → 자체 영상으로 교체
- **비용:** Grok API 3~4회 = ~$12 (~17,000원)

### 1-2. 통신판매업 신고 완료
- **작업:** 정부24 신고 → 2~3영업일 후 신고번호 → footer에 추가

### 1-3. 보증보험 가입
- **작업:** 서울보증보험 마포지점 연락 → 이행보증보험 200만원

### 1-4. NHN KCP 카드사 심사 요청
- **전제:** 통신판매신고번호 footer 반영 + 사이트 완성
- **작업:** partner.kcp.co.kr → 카드사 등록 심사 신청

### 1-5. KG이니시스 실연동 마무리
- **작업:** 메일 확인 → 키파일 등록 → 포트원 실연동 채널 설정
